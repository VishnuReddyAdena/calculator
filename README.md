# calculator
A simple GUI calculator built with Python’s Tkinter library.
It supports basic arithmetic operations along with decimal numbers, clear, backspace, and plus/minus toggle.

-> Features

Numeric buttons (0–9)

Decimal support (. button)

Arithmetic operations:

+ Addition

- Subtraction

x Multiplication

/ Division

% Modulus

= button to evaluate results

AC button to clear input

<- button for backspace

+/- button to toggle positive/negative values

Error handling (e.g., division by zero shows "Error")

🛠️ Requirements

Python 3.x

Tkinter (usually comes pre-installed with Python)

-> How to Run

Clone or download this repository.

Navigate to the project folder.

Run the script:

python calculator.py


The calculator GUI will open.
